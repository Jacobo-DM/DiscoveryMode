# Vessels

A vessel is a form of holding.

In DiscoveryMode, the vessel is not only an object but a way of learning through relation: clay, sound, water, pattern, warmth, memory, weather, breakage, and return.

This thread begins with the clay pot, but does not end there. In one place the vessel may be clay; in another, basket, garden bed, gourd, shell, or another local form of holding.

## Root notes

- Perception is positional.
- A good vessel does not stop change. It gives change a place to be noticed.
- The inquiry is not pottery itself, but local forms of holding.
- Tools made in relation carry memory differently than tools merely acquired.
